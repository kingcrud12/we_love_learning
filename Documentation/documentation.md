# WLL - We love learning - Documentation

**Présentation**
La présente documentation centralise tout ce qu'il y a à savoir sur la plateforme WLL.

Elle est divisée en différentes docuemnts :

- un document functionnal_analysis.html (disponible via documentation/functional_analysis) : qui concentre le résultat de l'analyse fonctionnelle menée autour du projet

- un document conception.html (disponible via documentation/conception) : qui concentre toute conception menée autour du projet

- un document backend.html (disponible via documentation/backend) : qui informe sur tout ce qu'il y a à savoir sur le backend de la plateforme

- un document frontend.html (disponible via documentation/frontend) : qui informe sur toute la partie partie frontend du projet
